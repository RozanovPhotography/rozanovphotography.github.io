How to install Chrome Extension

  • 1. Download the zip file using your browser and unzip it to the folder you want it to be stored in, like "Documents". Then open Extension Settings in your Chrome Browser, located at "chrome://extensions".

     

     

  • 2. In the top right corner, turn on the "Developer mode" on.

     

     

  • 3. In the top left corner, press "Load unpacked" and select the "Chrome Extension Michael Rozanov" folder.

     

     

  • 4. You can now turn off the "Developer mode".

     

     

  • 5. In the top right corner, press the Chrome extensions icon, find "Michael Rozanov" and pin it.

     

     

  • 6. The extension is successfully installed. Enjoy.
